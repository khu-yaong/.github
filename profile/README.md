# YAONG | 야구보러 옹기종기

## ⚙️ 기술 스택
- **Frontend** : React, TypeScript
- **Backend** : Spring
- **Database** : RDS(MySQL)
- **Server** : AWS EC2
- **Storage** : AWS S3
